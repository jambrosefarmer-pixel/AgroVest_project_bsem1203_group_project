#  AgroVest

AgroVest is a web-based agricultural investment platform designed to connect investors with farmers seeking financial support for agricultural projects. The platform allows users to explore investment opportunities, monitor investments, and contribute towards sustainable agricultural development through a secure and user-friendly system.


##  Project Overview

Agriculture plays a vital role in economic growth and food security. However, many farmers struggle to access adequate funding to expand and sustain their operations. AgroVest addresses this challenge by providing a digital platform that enables investors to support agricultural projects while receiving transparent updates on the performance of their investments.

The platform promotes financial inclusion, empowers local farmers, and contributes to sustainable development.


##  Features

- User registration and authentication
- Secure login and account management
- Browse agricultural investment opportunities
- Investment tracking dashboard
- Farmer project listings
- Investment status monitoring
- Responsive and user-friendly interface
- Role-based access control
